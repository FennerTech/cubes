[en] Visitor Counter from besucherzaehler-kostenlos.de

This website uses an external counter to track the number of page views. For this, a JavaScript is loaded from an external website. The server of besucherzaehler-kostenlos.de stores the IP address of the access anonymously and temporarily in a LOG file, which is regularly and irrevocably deleted.

To ensure the proper functioning of the counter, it also saves a so-called session cookie on the visitor's computer. This is usually deleted by the browser once it is closed. This cookie does not store any personal information. It only contains the information of the accessed domain and a boolean tag (true/false) to mark the visitor as already counted.

No personal or personally identifiable data is collected by the visitor counter. Tracking or assignment of accesses is not possible at any time. Special thanks to putzfrau-agentur.de, whose support makes this free service possible.

The level is calculated based on the total number of cubes you have created. It uses a logarithmic scale to ensure balanced progression. The formula is:
Das Level wird basierend auf der Gesamtzahl der erstellten Würfel berechnet. Es verwendet eine logarithmische Skala, um einen ausgewogenen Fortschritt zu gewährleisten. Die Formel lautet: Level = Math.floor(Math.log2(totalCubes + 1))
